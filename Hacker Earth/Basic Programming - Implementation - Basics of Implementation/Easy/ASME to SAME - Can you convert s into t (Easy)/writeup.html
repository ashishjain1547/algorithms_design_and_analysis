You are given 2 strings s and t both containing only lowercase English characters with s containing some ? as well. You can swap characters at positions i and j in the string s (j > i) any number of times and can also replace any ? with any lowercase English character. Can you convert s into t ?

Input Format

The first line contains an integer T denoting the number of test cases. 

The first line of each test case contains an integer N, the length of the strings.

The second line of each test case contains N lowercase English characters with some ? (possibly 0) denoting string s.

The third line of each test case contains N lowercase English characters denoting string t.

Output Format

For each test case, print 'Yes' (without quotation marks) if it is possible to convert s into t. Otherwise, print 'No' (without quotation marks). Print the output in a new line.

Constraints

1 <= T <= 100
1 <= N <= 10**5

Sample Output

Explanation

In the first sample, we can replace first 3 with characters a, m, a respectively to obtain the final string.

In the second sample, we can swap 3rd and 4th characters to get our final string.

In the third testcase, no matter what we do, we cannot convert s into t.

Initial Thoughts

1. The problem requires checking if we can transform string s into string t by swapping characters and replacing '?' characters.
2. Since we can swap characters freely, the order of characters in s does not matter.
3. We need to ensure that for every character in t, there are enough corresponding characters in s or '?' characters to replace.
4. We can use frequency counts to compare the characters in both strings.
5. If the count of any character in t exceeds the count in s plus the number of '?' characters, then the transformation is not possible.
6. Otherwise, it is possible to transform s into t.
7. We also need to check if there is some character in t that is not present in s (and otherwise: in s that is not in t) at all and there are no '?' characters in s to replace it with.

My Code

from collections import Counter

def can_convert(s, t):
    if len(s) != len(t):
        return "No"
    count_s = Counter(s)
    count_t = Counter(t)

    wildcards_s = count_s.get('?', 0)

    for char in count_t:
        if count_t[char] > count_s.get(char, 0):
            needed = count_t[char] - count_s.get(char, 0)
            wildcards_s -= needed
            if wildcards_s < 0:
                return "No"
        elif count_s.get(char, 0) > count_t[char]:
            return "No"
    return "Yes"

    

T = int(input())
for _ in range(T):
    N = int(input())
    s = input().strip()
    t = input().strip()
    print(can_convert(s, t))

def can_convert(s, t):
    from collections import Counter

    count_s = Counter(s)
    count_t = Counter(t)

    wildcards = count_s.get('?', 0)

    # Check each character needed in t
    for ch in count_t:
        available = count_s.get(ch, 0)
        needed = count_t[ch]

        if available >= needed:
            continue
        else:
            required_from_q = needed - available
            if required_from_q > wildcards:
                return "No"
            wildcards -= required_from_q

    return "Yes"


# ---- Driver code ----
T = int(input())
for _ in range(T):
    N = int(input())
    s = input().strip()
    t = input().strip()
    print(can_convert(s, t))

What the code does

  2. Counts how many ? characters are in s

  3. For each letter in t

  4. If all letters are satisfied ‚Üí print Yes
