‚úÖ Optimal Solution (Reverse and Compare)

python
S = input().strip()

if S == S[::-1]:
    print("YES")
else:
    print("NO")

üîç Explanation:

  • S[::-1] creates a reversed version of the string.

  • If the original and reversed strings are the same, it's a palindrome.

  • Time complexity: O(N)

  • Space complexity: O(N) (due to the reversed copy)

‚úÖ Alternative: Two-Pointer Approach (Saves Memory)

If you want O(1) space, you can avoid reversing the string:

python
S = input().strip()
is_palindrome = True

for i in range(len(S) // 2):
    if S[i] != S[-(i + 1)]:
        is_palindrome = False
        break

print("YES" if is_palindrome else "NO")

üîç Why This is More Optimal:

  • Uses only a flag and a loop.

  • No reversed copy of the string.

  • Time complexity: O(N), Space complexity: O(1)
