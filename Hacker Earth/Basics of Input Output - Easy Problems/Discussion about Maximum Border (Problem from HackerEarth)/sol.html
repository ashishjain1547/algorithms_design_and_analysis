Problem

You are given a table with n rows and m columns. Each cell is colored with white or black. Considering the shapes created by black cells, what is the maximum border of these shapes? Border of a shape means the maximum number of consecutive black cells in any row or column without any white cell in between.

A shape is a set of connected cells. Two cells are connected if they share an edge. Note that no shape has a hole in it.

Input format

The first line contains t denoting the number of test cases.
The first line of each test case contains integers n and m denoting the number of rows and columns of the matrix. Here, '#' represents a black cell and '.' represents a white cell. 
Each of the next n lines contains m integers.

Output format
Print the maximum border of the shapes.

✅ Problem Clarified (Based on Samples)

You're not being asked to detect connected shapes. You're asked for:

The maximum number of consecutive black cells (#) in any row or column across the entire matrix.

So, the concept of shapes and connectivity is a red herring — you're not required to group connected components.

✅ Actual Requirement (Based on Examples):

Given a grid of R x C, for each test case:

  • Find the longest sequence of # in any row or any column.

  • That sequence can be anywhere in the grid.

  • You're not restricted to connected components at all.

  • So no DFS/BFS is needed.

✅ Corrected and Optimized Approach

  1. Loop through every row:

    • Count max number of consecutive #s.

  2. Loop through every column:

    • Count max number of consecutive #s.

  3. Take the maximum from both row-wise and column-wise streaks.

✅ Python Code Implementation

python
def max_border(grid, R, C):
    max_streak = 0

    # Check row-wise
    for row in grid:
        count = 0
        for cell in row:
            if cell == '#':
                count += 1
                max_streak = max(max_streak, count)
            else:
                count = 0

    # Check column-wise
    for col in range(C):
        count = 0
        for row in range(R):
            if grid[row][col] == '#':
                count += 1
                max_streak = max(max_streak, count)
            else:
                count = 0

    return max_streak

# Driver code
T = int(input())
for _ in range(T):
    R, C = map(int, input().split())
    grid = [input().strip() for _ in range(R)]
    print(max_border(grid, R, C))

✅ Time and Space Complexity

  • Time Complexity: O(R × C)

  • Space Complexity: O(1) (excluding input)
