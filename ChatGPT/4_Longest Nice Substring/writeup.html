Problem: Longest Nice Substring

A string is considered "nice" if it contains equal numbers of vowels and consonants, and all characters are lowercase English letters.

You are given a lowercase string s. Find the length of the longest contiguous "nice" substring.

Input

A single string s consisting of only lowercase English letters.
1 <= len(s) <= 1000

Output

Print the length of the longest "nice" substring in s.

Example

Input

abcdeiou

Output

6

Certainly! Here's a straightforward solution using brute-force with optimizations, suitable for a 4/10 difficulty level.

‚úÖ Solution Explanation

  1. Iterate over all possible substrings of the given string s.

  2. For each substring:

    • Count the number of vowels and consonants.

    • If both are equal, it's a "nice" substring.

    • Track the maximum length found.

Since the max length of s is 1000, a brute-force approach with substring checks is still acceptable for small inputs.

üß† Python Code

python
def is_vowel(ch):
    return ch in 'aeiou'

def longest_nice_substring(s):
    n = len(s)
    max_len = 0

    for i in range(n):
        vowels = 0
        consonants = 0

        for j in range(i, n):
            if is_vowel(s[j]):
                vowels += 1
            else:
                consonants += 1

            if vowels == consonants:
                max_len = max(max_len, j - i + 1)

    return max_len

# Example usage
s = input().strip()
print(longest_nice_substring(s))

üìà Time Complexity

  • Worst-case: O(n¬≤) ‚Äî due to checking all substrings.

  • But inner loop operations are O(1), making it acceptable for n ‚â§ 1000.

‚úÖ Optimized Approach: Prefix Sum + Hash Map

We treat:

  • vowel as +1

  • consonant as -1

We compute a running sum. If the same value has occurred before, then the substring in between has an equal number of vowels and consonants.

üß† Python Code (Optimized):

python
def is_vowel(ch):
    return ch in 'aeiou'

def longest_nice_substring(s):
    n = len(s)
    prefix_sum = 0
    index_map = {0: -1}  # sum -> earliest index
    max_len = 0

    for i in range(n):
        if is_vowel(s[i]):
            prefix_sum += 1
        else:
            prefix_sum -= 1

        if prefix_sum in index_map:
            max_len = max(max_len, i - index_map[prefix_sum])
        else:
            index_map[prefix_sum] = i

    return max_len

# Example usage
s = input().strip()
print(longest_nice_substring(s))

üìà Time and Space Complexity:

  • Time: O(n)

  • Space: O(n) (for the hashmap)

üîÅ Problem Recap

You are given a string of lowercase letters, and you must find the length of the longest substring that contains an equal number of vowels and consonants.

‚úÖ Key Insight:

We convert this into a running balance problem:

  • Every vowel is treated as +1

  • Every consonant is treated as -1

Now the goal becomes:
üëâ Find the longest substring where the total balance is zero, i.e., the number of vowels and consonants are equal.

    • So we calculate i - earlier_index and update the max_len accordingly.
